## üí° **Tips for Best Screenshots**

1. **Use a consistent browser window size** (1920x1080 recommended)
2. **Clear browser cache** before starting
3. **Use real data** - select counties near cities for air quality data
4. **Wait for loading** - make sure all data is loaded before capturing
5. **Full page screenshots** - use browser extensions for dashboards
6. **High resolution** - capture at 100% zoom or higher
7. **Clean state** - close any browser notifications or popups
8. **Consistent time** - capture all screenshots in one session for consistency
